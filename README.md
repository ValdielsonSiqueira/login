# @FIAP/account (Login)

## Descrição

Microfrontend responsável pela autenticação e gestão de conta do usuário, incluindo as telas de Login e Cadastro.

## Tecnologias

- **Framework**: React.
- **Linguagem**: TypeScript.
- **Arquitetura**: Single-SPA.
- **Design System**: [`@valoro/ui`](https://www.npmjs.com/package/@valoro/ui).
- **Build Tool**: Webpack.

## Pré-requisitos

- **Node.js**: Versão LTS (v20+ recomendada).
- **Gerenciador de Pacotes**: pnpm.

## Como Rodar

1. Instale as dependências:

   ```bash
   pnpm install
   ```

2. Inicie o servidor:
   ```bash
   pnpm start
   ```

## Funcionalidades

- **Login**: Autenticação de usuários.
- **Cadastro**: Registro de novos usuários.
- **Recuperação de Senha**: Fluxo de "Esqueci minha senha".
